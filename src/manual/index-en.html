Creative Writing Program

Creative Writing is a writing exercise program for writers that want to experiment new literary production paradigm. The idea to develop this software came up after creative writing activity performed in a Paraguaçuense Academy of Letters meeting. Deck cards with different images were put on the table with images face down, each people randomly took five cards and wrote a text based on ideas suggested by card images. Creative Writing Software is a digital version of this exercise.

How it works

The interface contains two panels: right panel is a text editor in which user can digit, left panel displays images randomly chosen from image library. First, click File ↠ new to start a new work. In dialog window, digit a name to your file or click BROWSER to select a directory to save your file in xml format. This file contains your text and used images list. Select image source, that is, a directory containing images in png or jpeg format. You can select one or more directories. Check Default library to use default image library stored in this Creative Writing Software (in this way, you need not add other directories as image source, although you can). Click Add to add other directories as image source. If you want remove an added directory, click this directory name, then click Remove button. Click OK to close dialog window and starts your text.

A random image will be shown in left panel, write your text based on ideas suggested by image. You can write a word, a phrase, a paragraph or a larger text. Click NEXT button to display a new image and to continue your text. Click PREVIOUS to see previous images. Select File ↠ save to save your work in xml file format. To save in txt file format, click File ↠ save as txt. If you want to open an existing file, click File ↠ open, you can open only xml files.
