Programa Escrita Criativa

Escrita Criativa é um programa de exercício de escrita cujo público alvo são escritores em geral que queiram experimentar novos paradigmas de produção literária. A ideia do desenvolvimento deste software surgiu após uma atividade de escrita criativa realizada em uma das reuniões da Academia Paraguaçuense de Letras. Nesta atividade, cartas de baralho com diferentes imagens foram distribuídas pela mesa com as imagens viradas para baixo, cada pessoa pegou aleatoriamente cinco cartas e criou um texto a partir das ideias sugeridas pelos desenhos nas cartas. Escrita Criativa é uma versão digital deste exercício.

Como funciona

A interface do programa é composta por dois painéis: o painel da direita é um editor de texto onde o usuário pode digitar, o painel da esquerda exibe imagens aleatórias sugerindo ideias para a criação do texto. Inicialmente, deve-se clicar em Arquivo ↠ novo para iniciar um novo trabalho. Na janela que abrir, digite um nome para seu arquivo ou clique em BROWSER para selecionar uma pasta onde guardar seu arquivo, este será salvo no formato xml, contendo seu texto e um histórico das imagens utilizadas como inspiração para a escrita do texto. Em seguida, selecione a fonte de suas imagens, ou seja, a pasta que contém arquivos de imagem nos formatos png e/ou jpeg. É possível selecionar uma ou mais pastas como fonte de imagens. Marque a caixa Biblioteca padrão para usar a biblioteca de imagens que vem junto com o programa (neste caso, não é necessário adicionar outras pastas como fonte de imagens, embora seja possível) . Clique em ADICIONAR para selecionar outras pastas de seu computador como fonte de imagens. Se quiser remover uma pasta selecionada anteriormente, clique no nome da pasta e em seguida no botão Remover. Clique em OK para fechar a janela de diálogo e iniciar seu texto.

Uma imagem aleatória será exibida no painel esquerdo, comece a escrever seu texto baseado nas impressões provocadas pela imagem. Pode ser uma palavra, uma frase, um parágrafo ou um texto maior. Clique no botão PRÓXIMA para exibir uma nova imagem e continuar seu texto. Clique em ANTERIOR para rever as imagens anteriores. Para salvar seu trabalho, clique em Arquivo ↠ salvar, será salvo o arquivo em xml. Para salvar uma versão do texto no formato txt, clique em Arquivo ↠ salvar como txt. Se quiser abrir um trabalho salvo anteriormente, clique em Arquivo ↠ abrir, apenas arquivos xml podem ser abertos.
